# Public Key Cryptography and RSA Report

This repository contains a chapter report on **public key cryptography and RSA**, covering the mathematical background, ElGamal, RSA trapdoor permutation, the weaknesses of textbook RSA, secure RSA padding, and the Bleichenbacher attack.

The project includes two parallel versions of the report:
- an English version
- a Vietnamese version

## Repository Structure

- `report/`: English LaTeX source and rendered PDF
- `report-vi/`: Vietnamese LaTeX source and rendered PDF
